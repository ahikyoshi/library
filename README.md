## Установка и запуск

> Нужно будет две консоли для запуска лайв сервера React и отдельный для express

1. К1 cd front && npm ci
2. K1 npm run server
3. K2 cd server && npm ci
4. K2 npm run server
5. Зайди в браузере на localhost:3000
